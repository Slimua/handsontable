Handsontable

MathJax cell type

A MathJax cell can be used to display formulas and symbols inside your table.

This cell requires MathJax to be added in your page. Check out how to load and configure MathJax.

If you don't provide a syntax attribute, the value is considered normal text unless you use the delimiters configured on MathJax (e.g., \( and \( for TeX). Otherwise, the whole string is parsed as a TeX, MathML or AsciiMath value
