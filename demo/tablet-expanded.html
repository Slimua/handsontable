Handsontable

Handsontable is a minimalistic approach to Excel-like table editor in HTML & jQuery

Tablet support

Handsontable itself has a basic support of touchscreen devices (like iPad) - you may try to touch the table cells on the main examples page from your tablet.

However it is not convenient to navigate between cells due to navigation buttons absence on the standard iPad/Android keyboard. You may include the Controller extension in order to show the navigation buttons near the table when its being edited

Try to touch the cell of the table below:

Code: 

  <script>
        var container = $("#exampleGrid1");

        container.handsontable({
            rows: 12,
            cols: 12,
            rowHeaders: true,
            colHeaders: true,
            minSpareCols: 1,
            minSpareRows: 1,
            Controller: true
        });

        var data = [
            ["", "Kia", "Nissan", "Toyota", "Honda"],
            ["2008", 10, 11, 12, 13],
            ["2009", 20, 11, 14, 13],
            ["2010", 30, 15, 12, 13]
        ];

        container.handsontable("loadData", data);

    </script>

Controller options

The following options may be specified when adding Controller plugin

OptionDescriptionDefault
ui Specifies the UI for the controller.
  • false - no UI will be created
  • 'top' - standard UI fixed on top right corner will be created
  • true or 'float' - standard UI which floats near table's right edge will be created
  • jQuery object - it will be linked with controller as UI. The elements with the following classes will be bound with correponding actions:
    • moveLeft
    • moveRight
    • moveUp
    • moveDown
    • edit
    • undo
    • redo
    • addRowAfter
    • addColumnAfter
    • prependRow
    • prependColumn
    • deleteRow
    • deleteColumn
 true
buttons Set of buttons for standard UI. May include the following:
  • 'add' - shows the 'Add row/column' buttons
  • 'delete' - shows the 'Delete row/column' buttons
  • 'undo' - shows the Undo/Redo buttons
  • 'move' - shows the navigation buttons
  • 'edit' - shows the Edit button
  • 'drag' - shows the 'move' handler, allows to drag controller UI. Requires 'jquery.draggable' plugin
  • 'expand' - shows the 'expand/collapse' button, allows to expand the table on the whole screen. Requires 'jquery.expage' plugin
all of them, but 'drag' and 'expand' - only if corresponding plugins are installed
showfor Specifies when the UI shall be shown:
  • 'tablet' - for tablets only. Will be hidden for desktop
  • 'expage' - when expanded. Will be usually hidden
  • 'tablet expage' - for tablets when expanded
 '' (empty string)

Example

Code: 

  <script>
        var container = $("#exampleGrid2");

        container.handsontable({
            rows: 5,
            cols: 5,
            Controller: {
                buttons: ['move', 'add'],
                ui: 'float'
                }
        });

        var data = [
            ["", "Kia", "Nissan", "Toyota", "Honda"],
            ["2008", 10, 11, 12, 13],
            ["2009", 20, 11, 14, 13],
            ["2010", 30, 15, 12, 13]
        ];

        container.handsontable("loadData", data);

    </script>

Standard controller may be resized - just specify font-size for .handsontable-controller class

<style>
        /* Makes controller two times bigger */
        .handsontable-controller { font-size: 20px; }
</style>

Also plug-in may use the following additional libraries - just include them to the page:

  • jquery.draggable - if you need to drag the controller on desktop. Drag support for touchscreen devices is already included into controller
  • jquery.expage - if you want to expand table on the whole screen.
