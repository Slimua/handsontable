    Author: Handsontable  
    License: Open source (GPL 3.0 or Commercial)

- **numbro.js** (handles numeric data)<br>
    Author: Benjamin Van Ryseghem<br>
    License: Open source (MIT)<br>

- **Pikaday** (displays a date picker)<br>

    Author: Denis Pushkarev<br>
