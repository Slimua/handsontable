## Directory structure

The source files for the Handsontable project are stored on GitHub in a mono repository.

```
/
├── dist                # Compiled files
├── docs                # Documentation files
├── src                 # Source files
├── test                # Automated tests
├── tools               # Tools and utilities
├── LICENSE
└── README.md
```
